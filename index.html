Features 

Linux Stable



The Westcoast Superfast EAS+ Kernel is based on Linux 4.4.X. And is always provided latest ALS whenever it is releases by Linux on kernel.org

Security


 

The Westcoast Superfast EAS+ Kernel is always updated with latest Security CVE Fixes from both Linux,Google as well as Qualcomm(released by CAF).

Excellence


 

The Westcoast Superfast EAS+ Kernel is made to deliver Great performance and Exceptional Battery Life with Latest Energy Aware Scheduling techniques.
